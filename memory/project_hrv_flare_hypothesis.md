Alaric's hypothesis: HRV (specifically RMSSD) may actually *elevate* in the days before a lupus flare, which is counterintuitive (most would expect a drop). This is the primary motivation behind the flare/HRV overlay feature on the pattern analysis page.

**Why:** Building clinical evidence for next rheumatologist. If HRV elevation precedes flares, it could serve as an early warning signal and has implications for understanding autonomic behavior in lupus.

**How to apply:** Design the visualization to make upward/downward trends clearly visible — aggregate trajectory lines, trend indicators, and possibly statistical summaries. Don't assume the hypothesis is correct; the tool should reveal the pattern either way.
